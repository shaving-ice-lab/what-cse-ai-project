# WeRSS 网页预览模块

这个模块提供了基于FastAPI的网页预览功能，用于展示微信公众号文章内容。

## 功能特性

### 1. 首页 (`/views/home`)
- 显示所有标签列表
- 支持分页浏览
- 显示每个标签的公众号数量和文章数量
- 美观的卡片式布局

### 2. 文章列表页 (`/views/articles`)
- 显示文章列表
- 支持关键词搜索
- 支持按发布时间和创建时间排序
- 支持分页浏览
- 显示文章的公众号、发布时间、阅读状态

### 3. 文章详情页 (`/views/article/{article_id}`)
- 显示文章完整内容
- 显示公众号信息
- 支持返回文章列表
- 响应式设计，适配移动端

## 技术实现

### 数据模型
- **Tags模型**: 标签分类信息
- **Articles模型**: 文章内容信息
- **Feed模型**: 公众号信息

### 模板引擎
使用项目内置的轻量级模板引擎 `core.lax.template_parser`：

- 变量替换: `{{variable}}`
- 条件判断: `{% if condition %}...{% endif %}`
- 循环结构: `{% for item in items %}...{% endfor %}`

### 响应式设计
- 移动端友好
- 现代化UI设计
- 渐变色彩主题
- 卡片式布局

## 路由结构

```
/views/home                    # 首页 - 标签列表
/views/tag/{tag_id}            # 标签详情页
/views/articles                # 文章列表页
/views/article/{article_id}    # 文章详情页
```

## 安装使用

1. 路由已集成到主应用 `web.py` 中
2. 重启应用后即可访问:
   - 首页: `http://localhost:8001/views/home`
   - 文章列表: `http://localhost:8001/views/articles`
