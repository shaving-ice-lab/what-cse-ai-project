## §4 公共基础知识课程内容

> 📊 **模块概览**
> - 总课时：约80课时
> - 涵盖：政治理论、法律知识、公文写作、经济管理、人文科技
> - 适用于事业单位、三支一扶等考试

### 4.1 政治理论课程（28课时）

#### 马克思主义哲学精讲（8课时）

- [ ] **唯物论（2课时）**
  - [ ] 第1课：物质与意识 - 物质的定义与形态、意识的本质与能动作用、物质与意识的辩证关系
  - [ ] 第2课：运动与规律 - 运动与静止、规律的客观性、主观能动性与客观规律性

- [ ] **辩证法（2课时）**
  - [ ] 第3课：联系与发展 - 联系的普遍性、客观性、多样性、发展的实质与过程
  - [ ] 第4课：三大规律 - 对立统一规律、质量互变规律、否定之否定规律

- [ ] **认识论（2课时）**
  - [ ] 第5课：实践与认识 - 实践的本质与形式、认识的来源与过程、实践与认识的辩证关系
  - [ ] 第6课：真理与价值 - 真理的绝对性与相对性、价值观与价值评价、真理与价值的统一

- [ ] **唯物史观（2课时）**
  - [ ] 第7课：社会存在与社会意识 - 社会存在的构成、社会意识的相对独立性
  - [ ] 第8课：人民群众与历史发展 - 人民群众是历史创造者、个人在历史中的作用

#### 毛泽东思想精讲（4课时）

- [ ] 第1课：毛泽东思想概述 - 形成与发展、科学体系与主要内容、活的灵魂
- [ ] 第2课：新民主主义革命理论 - 革命总路线、三大法宝、农村包围城市道路
- [ ] 第3课：社会主义改造理论 - 过渡时期总路线、社会主义改造道路
- [ ] 第4课：社会主义建设探索 - 正确处理人民内部矛盾、社会主义建设总路线

#### 中国特色社会主义理论精讲（6课时）

- [ ] **邓小平理论（2课时）**
  - [ ] 第1课：理论核心内容 - 解放思想、实事求是、社会主义本质、社会主义初级阶段理论
  - [ ] 第2课：改革开放理论 - 改革的性质与目的、对外开放格局

- [ ] **三个代表重要思想与科学发展观（2课时）**
  - [ ] 第3课：三个代表重要思想 - 核心内容、历史地位
  - [ ] 第4课：科学发展观 - 科学内涵、基本要求

- [ ] **习近平新时代中国特色社会主义思想（2课时）**
  - [ ] 第5课：核心要义 - "八个明确"、"十四个坚持"
  - [ ] 第6课：重要论述专题 - 新发展理念、全面深化改革、全面依法治国、生态文明建设

#### 党史党建精讲（6课时）

- [ ] **党的发展历程（2课时）**
  - [ ] 第1课：新民主主义革命时期（1921-1949） - 建党初期、土地革命、抗日战争、解放战争
  - [ ] 第2课：社会主义革命和建设时期（1949-1978） - 过渡时期、社会主义建设探索

- [ ] **改革开放新时期（2课时）**
  - [ ] 第3课：改革开放历程 - 重要会议与决策、发展阶段与成就
  - [ ] 第4课：新时代党的建设 - 全面从严治党、党的建设总要求

- [ ] **党的重要会议（2课时）**
  - [ ] 第5课：历次党代会主要内容
  - [ ] 第6课：重要历史决议与文献

#### 时政热点专题（4课时）

- [ ] 第1课：国内时政要闻 - 重要会议精神、政策法规动态
- [ ] 第2课：国际形势与外交 - 国际局势变化、中国外交政策
- [ ] 第3课：经济社会发展 - 经济政策措施、民生保障政策
- [ ] 第4课：科技文化教育 - 科技创新成就、文化教育发展

### 4.2 法律知识课程（34课时）

#### 法理学精讲（2课时）

- [ ] 第1课：法的本质与特征 - 法的概念、法的特征、法的作用
- [ ] 第2课：法的运行 - 法的制定、法的实施、法律关系

#### 宪法精讲（6课时）

- [ ] **宪法基本理论（2课时）**
  - [ ] 第1课：宪法概述 - 宪法的地位与作用、宪法的发展历程、宪法修改
  - [ ] 第2课：国家基本制度 - 人民民主专政、人民代表大会制度、民族区域自治制度、基层群众自治制度

- [ ] **公民权利与义务（2课时）**
  - [ ] 第3课：公民基本权利 - 政治权利与自由、人身自由权利、社会经济权利、文化教育权利
  - [ ] 第4课：公民基本义务 - 维护国家统一、遵守宪法法律、依法纳税、服兵役

- [ ] **国家机构（2课时）**
  - [ ] 第5课：中央国家机关 - 全国人大及常委会、国家主席、国务院、中央军委、监察委员会、最高人民法院、最高人民检察院
  - [ ] 第6课：地方国家机关 - 地方各级人大、地方各级人民政府、地方监察机关、司法机关

#### 民法典精讲（10课时）

- [ ] **民法总则（2课时）**
  - [ ] 第1课：民事主体 - 自然人、法人、非法人组织
  - [ ] 第2课：民事行为与代理 - 民事法律行为、代理制度、民事责任

- [ ] **物权编（2课时）**
  - [ ] 第3课：物权通则与所有权 - 物权的种类、所有权取得与消灭、共有
  - [ ] 第4课：用益物权与担保物权 - 土地承包经营权、建设用地使用权、抵押权、质权、留置权

- [ ] **合同编（2课时）**
  - [ ] 第5课：合同通则 - 合同订立、合同效力、合同履行与违约
  - [ ] 第6课：典型合同 - 买卖合同、租赁合同、借款合同、劳务合同

- [ ] **人格权与侵权责任（2课时）**
  - [ ] 第7课：人格权编 - 生命权、身体权、健康权、姓名权、肖像权、名誉权、荣誉权、隐私权与个人信息保护
  - [ ] 第8课：侵权责任编 - 侵权责任构成、特殊侵权责任、损害赔偿

- [ ] **婚姻家庭与继承（2课时）**
  - [ ] 第9课：婚姻家庭编 - 结婚条件与程序、夫妻关系、离婚制度、收养制度
  - [ ] 第10课：继承编 - 法定继承、遗嘱继承与遗赠、遗产处理

#### 刑法精讲（6课时）

- [ ] **刑法总则（2课时）**
  - [ ] 第1课：犯罪论 - 犯罪构成要件、正当防卫与紧急避险、犯罪形态
  - [ ] 第2课：刑罚论 - 刑罚种类、量刑制度、刑罚执行

- [ ] **刑法分则常考罪名（4课时）**
  - [ ] 第3课：侵犯公民人身权利罪 - 故意杀人罪、故意伤害罪、强奸罪、猥亵罪、非法拘禁罪
  - [ ] 第4课：侵犯财产罪 - 盗窃罪、抢劫罪、抢夺罪、诈骗罪、敲诈勒索罪、侵占罪、挪用资金罪
  - [ ] 第5课：贪污贿赂罪 - 贪污罪、受贿罪、挪用公款罪、行贿罪
  - [ ] 第6课：渎职罪与其他常考罪名 - 滥用职权罪、玩忽职守罪、危害公共安全罪、妨害社会管理秩序罪

#### 行政法精讲（6课时）

- [ ] **行政法基础（2课时）**
  - [ ] 第1课：行政法基本原则 - 合法性原则、合理性原则、程序正当原则、诚实信用原则
  - [ ] 第2课：行政主体与相对人 - 行政主体的种类、行政授权与委托、公务员制度

- [ ] **行政行为（2课时）**
  - [ ] 第3课：具体行政行为 - 行政许可、行政处罚、行政强制、行政征收
  - [ ] 第4课：行政程序 - 行政决定程序、行政听证程序、行政公开制度

- [ ] **行政救济（2课时）**
  - [ ] 第5课：行政复议 - 复议范围与管辖、复议程序、复议决定
  - [ ] 第6课：行政诉讼与国家赔偿 - 行政诉讼受案范围、行政诉讼程序、国家赔偿制度

#### 其他法律精讲（4课时）

- [ ] 第1课：劳动与社会保障法 - 劳动合同法、社会保险法
- [ ] 第2课：知识产权法 - 著作权法、专利法、商标法
- [ ] 第3课：经济法 - 消费者权益保护法、反不正当竞争法、反垄断法
- [ ] 第4课：诉讼法 - 民事诉讼法基础、刑事诉讼法基础

### 4.3 公文写作课程（11课时）

#### 公文格式规范（3课时）

- [ ] 第1课：公文构成要素 - 版头（发文机关标志、发文字号等）、主体（标题、主送机关、正文、附件等）、版记（抄送机关、印发日期等）
- [ ] 第2课：公文版式规范 - 页面设置、字体字号、行距版心
- [ ] 第3课：公文格式常见错误 - 格式错误类型、格式审核要点

#### 常用文种精讲（6课时）

- [ ] 第1课：决定与决议 - 决定的写法、决议的写法
- [ ] 第2课：通知与通报 - 各类通知写法、各类通报写法
- [ ] 第3课：报告与请示 - 报告的类型与写法、请示的写法与注意事项
- [ ] 第4课：批复与函 - 批复的写法、各类函的写法
- [ ] 第5课：意见与纪要 - 意见的写法、会议纪要的写法
- [ ] 第6课：事务性文书 - 计划与总结、简报与调查报告、讲话稿与演讲稿

#### 公文处理流程（2课时）

- [ ] 第1课：公文办理 - 收文办理流程、发文办理流程
- [ ] 第2课：公文管理 - 公文归档、公文销毁

### 4.4 其他知识（7课时）

#### 经济管理课程（2课时）

- [ ] 第1课：宏微观经济学基础 - 宏观经济指标、微观经济理论
- [ ] 第2课：管理学基础 - 管理职能、管理方法

#### 人文历史课程（3课时）

- [ ] 第1课：中国历史 - 古代史重要知识点、近现代史重要事件
- [ ] 第2课：文学艺术 - 中国古典文学、中外文学名著
- [ ] 第3课：传统文化 - 传统节日与习俗、文化遗产与非遗

#### 科技地理课程（2课时）

- [ ] 第1课：科技常识 - 物理化学基础、生物医学常识、信息技术发展
- [ ] 第2课：地理常识 - 中国地理、世界地理
