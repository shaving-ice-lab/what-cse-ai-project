## §2 申论课程内容

> 📊 **模块概览**
> - 总课时：约120课时
> - 涵盖五大题型：归纳概括、提出对策、综合分析、贯彻执行、文章写作
> - 教学形式：方法精讲 + 真题演练 + 范文赏析 + 批改点评

### 2.1 申论导学课程（6课时）

- [ ] **申论考试概述（2课时）**
  - [ ] 第1课：申论考试性质与特点 - 申论与行测的区别、国考与省考申论差异、评分标准详解
  - [ ] 第2课：申论备考规划 - 备考阶段划分、各阶段学习重点、时间分配建议

- [ ] **申论作答方法论（2课时）**
  - [ ] 第3课：材料阅读方法 - 精读与略读技巧、关键信息标注法、段落层次划分
  - [ ] 第4课：答案要点提炼 - 关键词识别、要点合并同类项、逻辑框架搭建

- [ ] **申论材料类型解读（2课时）**
  - [ ] 第5课：案例型材料解读 - 正面案例提取经验、反面案例分析教训、对比案例辩证分析
  - [ ] 第6课：观点型/数据型材料解读 - 观点归纳与提炼、数据材料分析技巧、混合材料整合

### 2.2 归纳概括课程（18课时）

#### 归纳概括方法精讲（10课时）

- [ ] **归纳概括基础方法（3课时）**
  - [ ] 第1课：题干审题技巧 - 作答对象识别、作答范围判定、字数与格式要求
  - [ ] 第2课：要点提取方法 - 直接摘抄法、概括归纳法、推理引申法
  - [ ] 第3课：答案加工技巧 - 分类整理、合并同类项、逻辑排序

- [ ] **归纳概括题型分类（4课时）**
  - [ ] 第4课：概括问题类 - 问题的表现形式、问题的层次划分、典型真题精讲
  - [ ] 第5课：概括原因类 - 原因的多角度分析、因果关系链条梳理、典型真题精讲
  - [ ] 第6课：概括影响/意义类 - 积极与消极影响、短期与长期影响、不同主体影响
  - [ ] 第7课：概括做法/经验类 - 成功经验提炼、可复制性分析、典型真题精讲

- [ ] **归纳概括进阶（3课时）**
  - [ ] 第8课：概括特点/特征类 - 特点的提炼角度、特点的表述方式
  - [ ] 第9课：概括变化/趋势类 - 时间线梳理、变化对比分析
  - [ ] 第10课：概括争议/观点类 - 不同观点梳理、争议焦点概括

#### 各类归纳概括题型实战（8课时）

- [ ] **国考真题实战（4课时）**
  - [ ] 第1课：国考地市级归纳概括真题
  - [ ] 第2课：国考省部级归纳概括真题
  - [ ] 第3课：难点题目深度解析
  - [ ] 第4课：高分答案对比分析

- [ ] **省考真题实战（4课时）**
  - [ ] 第5课：联考归纳概括真题
  - [ ] 第6课：各省特色题型解析
  - [ ] 第7课：易错点总结
  - [ ] 第8课：模拟演练与点评

### 2.3 提出对策课程（16课时）

#### 对策来源分析（8课时）

- [ ] **直接对策来源（4课时）**
  - [ ] 第1课：材料直接对策 - 已采取的措施、专家学者建议、群众呼声与期盼
  - [ ] 第2课：问题推导对策 - 针对问题表现/原因提对策、问题-对策对应关系
  - [ ] 第3课：经验借鉴对策 - 成功案例经验提炼、国内外先进经验借鉴、经验本土化改造
  - [ ] 第4课：教训反推对策 - 失败教训分析、反向思维提对策、避免重蹈覆辙

- [ ] **对策完善与优化（4课时）**
  - [ ] 第5课：对策的可行性分析 - 经济、技术、政策可行性、社会接受度
  - [ ] 第6课：对策的针对性检验 - 对策与问题的对应、避免答非所问和泛泛而谈
  - [ ] 第7课：对策的可操作性提升 - 主体明确、措施具体、方式方法清晰
  - [ ] 第8课：对策表述规范 - 条目化表述、概括+具体展开、字数把控

#### 对策维度框架（8课时）

- [ ] **常用对策维度（4课时）**
  - [ ] 第1课：主体维度 - 政府层面、社会层面、个人层面
  - [ ] 第2课：要素维度 - 人（人才）、财（资金）、物（设施）、技术（科技）
  - [ ] 第3课：环节维度 - 事前（预防）、事中（监管）、事后（评估）
  - [ ] 第4课：手段维度 - 法律手段、行政手段、经济手段、教育手段

- [ ] **对策框架综合应用（4课时）**
  - [ ] 第5课：单一主体对策框架
  - [ ] 第6课：多主体协同对策框架
  - [ ] 第7课：系统性对策框架
  - [ ] 第8课：真题对策框架分析

### 2.4 综合分析课程（18课时）

#### 各类分析题型精讲（18课时）

- [ ] **词句理解分析（5课时）**
  - [ ] 第1课：词语理解题解题思路 - 词语的字面含义、语境含义、深层含义
  - [ ] 第2课：句子理解题解题思路 - 句子的表层含义、深层含义、引申含义
  - [ ] 第3课：答案结构组织 - 含义+分析+总结/对策、要点层次安排
  - [ ] 第4课：国考词句理解真题精讲
  - [ ] 第5课：省考词句理解真题精讲

- [ ] **观点评价分析（5课时）**
  - [ ] 第6课：观点评价题解题思路 - 观点识别与概括、多角度评价、评价依据
  - [ ] 第7课：单一观点评价 - 完全正确型观点、完全错误型观点
  - [ ] 第8课：辩证观点评价 - 既对又错型观点、各有道理型观点
  - [ ] 第9课：多观点比较分析 - 观点异同分析、综合评价与建议
  - [ ] 第10课：观点评价真题精讲

- [ ] **原因/影响分析（4课时）**
  - [ ] 第11课：原因分析题解题思路 - 直接与根本原因、主观与客观原因、原因层次递进
  - [ ] 第12课：影响分析题解题思路 - 积极与消极影响、直接与间接影响、当前与长远影响
  - [ ] 第13课：原因分析真题精讲
  - [ ] 第14课：影响分析真题精讲

- [ ] **比较分析与启示分析（4课时）**
  - [ ] 第15课：比较分析题解题思路 - 异同点比较框架、比较维度选择、比较结论提炼
  - [ ] 第16课：启示分析题解题思路 - 启示的来源、启示的转化、启示的表述规范
  - [ ] 第17课：比较分析真题精讲
  - [ ] 第18课：启示分析真题精讲

### 2.5 贯彻执行课程（32课时）

#### 公文格式规范（6课时）

- [ ] **公文基础格式（3课时）**
  - [ ] 第1课：标题写法 - 完整式标题、省略式标题、新闻类标题
  - [ ] 第2课：称谓与落款 - 主送机关写法、抄送机关写法、发文日期格式
  - [ ] 第3课：正文结构 - 开头（背景目的）、主体（核心内容）、结尾（号召呼吁）

- [ ] **各类公文格式对比（3课时）**
  - [ ] 第4课：上行文格式要求 - 请示、报告
  - [ ] 第5课：下行文格式要求 - 通知、批复、意见
  - [ ] 第6课：平行文格式要求 - 函、议案

#### 各类公文写作（26课时）

##### 讲话稿类（4课时）

- [ ] 第1课：讲话稿基本结构 - 开场白（问候、背景）、正文（核心内容、层次分明）、结尾（号召、祝福）
- [ ] 第2课：不同场合讲话稿 - 会议讲话稿、活动致辞、动员讲话
- [ ] 第3课：讲话稿语言风格 - 口语化表达、感染力与号召力、节奏与停顿
- [ ] 第4课：讲话稿真题精讲

##### 宣传倡议类（4课时）

- [ ] 第1课：宣传材料写作 - 宣传手册/海报、宣传标语/口号、宣传文案
- [ ] 第2课：倡议书写作 - 倡议书格式与结构、倡议内容组织、倡议号召技巧
- [ ] 第3课：公开信写作 - 公开信特点、公开信结构、情感表达技巧
- [ ] 第4课：宣传倡议类真题精讲

##### 报告总结类（5课时）

- [ ] 第1课：工作报告写作 - 工作报告结构、成绩与问题并述、数据与案例支撑
- [ ] 第2课：调研报告写作 - 调研背景与目的、调研发现与分析、调研建议与结论
- [ ] 第3课：工作总结写作 - 总结结构（回顾+分析+展望）、经验提炼、问题剖析
- [ ] 第4课：情况汇报写作 - 汇报结构、重点突出、简明扼要
- [ ] 第5课：报告总结类真题精讲

##### 方案计划类（5课时）

- [ ] 第1课：工作方案写作 - 方案结构（目标+措施+保障）、措施具体可操作、责任分工明确
- [ ] 第2课：活动方案写作 - 活动目的与主题、活动内容与流程、活动保障与预案
- [ ] 第3课：整改方案写作 - 问题分析、整改措施、整改时限
- [ ] 第4课：计划/规划写作 - 年度计划/工作计划、发展规划、时间节点安排
- [ ] 第5课：方案计划类真题精讲

##### 新闻传媒类（4课时）

- [ ] 第1课：新闻稿写作 - 新闻六要素、倒金字塔结构、新闻标题拟定
- [ ] 第2课：简报写作 - 简报结构与格式、简报内容选择、简报语言风格
- [ ] 第3课：导言/编者按写作 - 导言结构与功能、编者按写作技巧
- [ ] 第4课：新闻传媒类真题精讲

##### 其他文书（4课时）

- [ ] 第1课：提纲/要点写作 - 发言提纲、汇报要点、工作要点
- [ ] 第2课：短评/短文写作 - 短评结构、观点鲜明、论证有力
- [ ] 第3课：建议/意见写作 - 建议书格式、意见的表述、建议的针对性
- [ ] 第4课：其他文书真题精讲

### 2.6 文章写作课程（30课时）

#### 立意技巧（6课时）

- [ ] **审题立意基础（2课时）**
  - [ ] 第1课：题干分析法 - 关键词识别、题目类型判断、立意范围确定
  - [ ] 第2课：材料分析法 - 核心材料识别、主旨提炼、立意角度选择

- [ ] **立意提升技巧（2课时）**
  - [ ] 第3课：立意的高度提升 - 从个别到一般、从现象到本质、从问题到对策
  - [ ] 第4课：立意的角度创新 - 多角度思考、逆向思维、辩证思维

- [ ] **立意常见误区（2课时）**
  - [ ] 第5课：偏题跑题分析 - 偏离核心话题、立意过大或过小、立意空洞
  - [ ] 第6课：优秀立意案例分析

#### 标题拟定（3课时）

- [ ] 第1课：标题拟定原则 - 准确体现立意、简洁有力、富有文采
- [ ] 第2课：常用标题格式 - 对仗式标题、比喻式标题、引用式标题、设问式标题
- [ ] 第3课：优秀标题案例赏析 - 国考高分标题、省考高分标题、标题修改训练

#### 开头写法（5课时）

- [ ] **常用开头方式（2课时）**
  - [ ] 第1课：开门见山式 - 直接亮明观点、简洁明了
  - [ ] 第2课：引用名言式 - 名言选择原则、名言引用技巧、名言与主题衔接

- [ ] **进阶开头方式（2课时）**
  - [ ] 第3课：案例引入式 - 案例选择、案例概述、案例分析引出观点
  - [ ] 第4课：背景分析式 - 时代背景、政策背景、社会背景

- [ ] **开头写作实战（1课时）**
  - [ ] 第5课：开头写作实战 - 开头模板练习、开头修改优化、真题开头赏析

#### 分论点论证（8课时）

- [ ] **分论点设置（2课时）**
  - [ ] 第1课：分论点数量与结构 - 2-4个分论点、并列式/递进式/对比式
  - [ ] 第2课：分论点表述技巧 - 分论点的统一性、差异性、概括性

- [ ] **论证方法（3课时）**
  - [ ] 第3课：举例论证 - 案例选择原则、案例叙述技巧、案例分析技巧
  - [ ] 第4课：道理论证 - 引用名言警句、理论分析、因果分析
  - [ ] 第5课：对比论证 - 正反对比、古今对比、中外对比

- [ ] **论证段落写作（3课时）**
  - [ ] 第6课：段落结构组织 - 分论点句+阐释句+论据句+分析句+总结句、字数分配
  - [ ] 第7课：论证段落衔接 - 段落间过渡、逻辑连贯性
  - [ ] 第8课：分论点论证实战练习

#### 结尾写法（3课时）

- [ ] 第1课：常用结尾方式 - 总结全文式、号召呼吁式、展望未来式
- [ ] 第2课：进阶结尾方式 - 引用名言式、设问反问式、照应开头式
- [ ] 第3课：结尾写作实战 - 结尾模板练习、真题结尾赏析

#### 高分范文赏析（5课时）

- [ ] **国考高分范文（2课时）**
  - [ ] 第1课：省部级高分范文赏析
  - [ ] 第2课：地市级高分范文赏析

- [ ] **省考高分范文（2课时）**
  - [ ] 第3课：联考高分范文赏析
  - [ ] 第4课：各省特色范文赏析

- [ ] **范文写作技巧总结（1课时）**
  - [ ] 第5课：范文写作技巧总结 - 高分作文共性分析、得分点与失分点、备考建议
