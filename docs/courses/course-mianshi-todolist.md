## §3 面试课程内容

> 📊 **模块概览**
> - 总课时：约100课时
> - 涵盖：结构化面试、无领导小组讨论、面试礼仪
> - 教学形式：理论精讲 + 真题演练 + 模拟实战 + 点评指导

### 3.1 面试导学课程（6课时）

- [ ] **面试考试概述（2课时）**
  - [ ] 第1课：公务员面试形式与特点 - 结构化面试vs无领导、国考与省考差异、不同部门面试特点
  - [ ] 第2课：面试评分标准详解 - 综合分析、计划组织、人际沟通、应变、语言表达、举止仪表

- [ ] **面试备考策略（2课时）**
  - [ ] 第3课：面试备考规划 - 备考时间分配、各阶段训练重点、资料准备与使用
  - [ ] 第4课：面试答题方法论 - 审题技巧、答题框架搭建、时间分配原则

- [ ] **面试心态调适（2课时）**
  - [ ] 第5课：面试心理准备 - 常见心理障碍、紧张情绪克服、自信心建立
  - [ ] 第6课：面试状态调整 - 考前状态调整、考场应对技巧、突发情况处理

### 3.2 结构化面试课程（70课时）

#### 综合分析题精讲（20课时）

##### 社会现象类（6课时）

- [ ] **社会现象分析框架（2课时）**
  - [ ] 第1课：积极/正面现象分析 - 表态+原因/意义+启示/对策、推广可行性分析
  - [ ] 第2课：消极/负面现象分析 - 表态+原因+危害+对策、多主体分析视角

- [ ] **社会现象专题（2课时）**
  - [ ] 第3课：社会热点现象 - 网络治理/就业问题等
  - [ ] 第4课：基层治理现象 - 形式主义/官僚主义等

- [ ] **社会现象实战（2课时）**
  - [ ] 第5课：国考社会现象真题精讲
  - [ ] 第6课：省考社会现象真题精讲

##### 政策理解类（5课时）

- [ ] **政策分析框架（2课时）**
  - [ ] 第1课：政策出台背景分析 - 问题导向、目标导向
  - [ ] 第2课：政策内容与影响分析 - 政策积极意义、潜在问题与挑战、完善建议

- [ ] **政策类型专题（2课时）**
  - [ ] 第3课：民生政策分析 - 教育/医疗/养老等
  - [ ] 第4课：治理政策分析 - 环保/安全/监管等

- [ ] **政策理解真题精讲（1课时）**
  - [ ] 第5课：政策理解真题精讲

##### 名言警句类（4课时）

- [ ] **名言警句分析框架（2课时）**
  - [ ] 第1课：名言内涵解读 - 关键词分析、引申义阐释
  - [ ] 第2课：名言启示与实践 - 对工作的指导意义、具体实践路径

- [ ] **名言警句专题（2课时）**
  - [ ] 第3课：习近平总书记重要讲话
  - [ ] 第4课：传统文化经典名言

##### 哲理故事类（5课时）

- [ ] **哲理故事分析框架（2课时）**
  - [ ] 第1课：故事核心道理提炼 - 故事梗概、寓意阐释
  - [ ] 第2课：道理的多角度阐发 - 结合实际工作、结合个人成长

- [ ] **哲理故事专题（3课时）**
  - [ ] 第3课：寓言故事类
  - [ ] 第4课：历史典故类
  - [ ] 第5课：哲理故事真题精讲

#### 计划组织题精讲（18课时）

##### 调研类（5课时）

- [ ] **调研活动框架（2课时）**
  - [ ] 第1课：调研前期准备 - 调研目的明确、调研内容设计、调研对象选择
  - [ ] 第2课：调研实施与总结 - 调研方式方法、数据收集与分析、调研报告撰写

- [ ] **调研专题（2课时）**
  - [ ] 第3课：基层调研 - 走访入户/座谈会等
  - [ ] 第4课：专项调研 - 政策落实/问题排查等

- [ ] **调研类真题精讲（1课时）**
  - [ ] 第5课：调研类真题精讲

##### 宣传类（4课时）

- [ ] **宣传活动框架（2课时）**
  - [ ] 第1课：宣传活动策划 - 宣传目的与主题、宣传对象与渠道、宣传内容与形式
  - [ ] 第2课：宣传效果保障 - 多渠道联动、效果评估与反馈、长效机制建立

- [ ] **宣传类真题精讲（2课时）**
  - [ ] 第3课：政策宣传类
  - [ ] 第4课：公益宣传类

##### 活动策划类（5课时）

- [ ] **活动策划框架（2课时）**
  - [ ] 第1课：活动策划要素 - 活动目的与主题、活动时间与地点、活动对象与规模
  - [ ] 第2课：活动实施与保障 - 活动流程安排、人员分工、物资与经费保障、应急预案

- [ ] **活动策划专题（3课时）**
  - [ ] 第3课：培训类活动
  - [ ] 第4课：比赛/评选类活动
  - [ ] 第5课：节日/纪念日活动

##### 会议组织类（4课时）

- [ ] **会议组织框架（2课时）**
  - [ ] 第1课：会前准备 - 会议方案制定、参会人员通知、会议材料准备、会场布置
  - [ ] 第2课：会中与会后 - 会议流程把控、会议记录、会议纪要与落实

- [ ] **会议类型专题（2课时）**
  - [ ] 第3课：工作会议/总结会
  - [ ] 第4课：协调会/座谈会

#### 人际关系题精讲（12课时）

- [ ] **人际关系分析框架（3课时）**
  - [ ] 第1课：人际关系题审题技巧 - 人物关系识别、矛盾点定位、题目类型判断
  - [ ] 第2课：人际关系处理原则 - 工作为重、尊重理解、沟通协调、反思改进
  - [ ] 第3课：人际关系答题框架 - 表态+分析+解决+反思

- [ ] **人际关系专题（5课时）**
  - [ ] 第4课：与领导的关系处理 - 领导批评/误解、领导安排冲突、向领导汇报/请示
  - [ ] 第5课：与同事的关系处理 - 同事不配合、同事竞争/冲突、新老同事关系
  - [ ] 第6课：与下属的关系处理 - 下属消极怠工、下属越级汇报、下属矛盾调解
  - [ ] 第7课：与群众的关系处理 - 群众不理解/不配合、群众投诉/上访、群众服务
  - [ ] 第8课：多重人际关系处理

- [ ] **人际关系真题实战（4课时）**
  - [ ] 第9课：国考人际关系真题
  - [ ] 第10课：省考人际关系真题
  - [ ] 第11课：特殊情境人际关系
  - [ ] 第12课：综合人际关系真题

#### 应急应变题精讲（12课时）

- [ ] **应急应变分析框架（3课时）**
  - [ ] 第1课：应急应变题审题技巧 - 紧急情况识别、矛盾主次判断、资源条件分析
  - [ ] 第2课：应急处理原则 - 冷静应对、轻重缓急、灵活变通、善后总结
  - [ ] 第3课：应急应变答题框架 - 稳定情绪+解决问题+善后处理+总结反思

- [ ] **应急应变专题（5课时）**
  - [ ] 第4课：突发公共事件 - 自然灾害、公共卫生事件、社会安全事件
  - [ ] 第5课：工作突发状况 - 工作失误、设备故障、人员缺位
  - [ ] 第6课：群众矛盾冲突 - 群体性事件、个体冲突、舆情应对
  - [ ] 第7课：活动现场突发 - 会议/活动变故、接待/检查突发
  - [ ] 第8课：媒体舆论应对

- [ ] **应急应变真题实战（4课时）**
  - [ ] 第9课：国考应急应变真题
  - [ ] 第10课：省考应急应变真题
  - [ ] 第11课：铁路公安等执法岗位
  - [ ] 第12课：综合应急应变真题

#### 自我认知题精讲（4课时）

- [ ] **自我认知题基础（2课时）**
  - [ ] 第1课：自我认知题类型 - 自我介绍、优缺点分析、职业规划、求职动机
  - [ ] 第2课：自我认知答题技巧 - 真实与适度包装、岗位匹配性展示、避免套路化

- [ ] **自我认知真题精讲（2课时）**
  - [ ] 第3课：常规自我认知题
  - [ ] 第4课：创新型自我认知题

#### 情景模拟题精讲（4课时）

- [ ] **情景模拟答题方法（2课时）**
  - [ ] 第1课：情景模拟特点与要求 - 入情入境、语言转换、身份带入
  - [ ] 第2课：情景模拟答题技巧 - 开场/称呼、内容组织、语气语调

- [ ] **情景模拟专题（2课时）**
  - [ ] 第3课：劝说/劝导类情景
  - [ ] 第4课：解释/安抚类情景

### 3.3 无领导小组讨论课程（15课时）

#### 题型分类精讲（6课时）

- [ ] **开放式问题（2课时）**
  - [ ] 第1课：开放式问题特点与策略
  - [ ] 第2课：开放式问题真题演练

- [ ] **两难式问题（2课时）**
  - [ ] 第3课：两难式问题分析框架
  - [ ] 第4课：两难式问题真题演练

- [ ] **排序/资源分配问题（2课时）**
  - [ ] 第5课：排序问题分析框架
  - [ ] 第6课：资源分配问题策略

#### 角色策略（5课时）

- [ ] **无领导角色定位（1课时）**
  - [ ] 第1课：无领导角色定位 - 领导者/组织者、时间管理者、记录者、参与者

- [ ] **各角色策略（2课时）**
  - [ ] 第2课：领导者角色策略
  - [ ] 第3课：其他角色策略

- [ ] **角色选择与转换（2课时）**
  - [ ] 第4课：角色选择原则
  - [ ] 第5课：角色转换技巧

#### 讨论技巧（4课时）

- [ ] 第1课：发言技巧 - 首发言策略、插话技巧、总结发言
- [ ] 第2课：倾听与回应 - 积极倾听、有效回应、求同存异
- [ ] 第3课：冲突处理 - 意见分歧处理、气氛调节、推动共识
- [ ] 第4课：无领导实战演练

### 3.4 面试礼仪课程（9课时）

#### 着装礼仪（2课时）

- [ ] 第1课：面试着装原则 - 男士着装规范、女士着装规范、配饰与发型
- [ ] 第2课：着装误区与建议 - 常见着装误区、不同季节着装、特殊岗位着装

#### 仪态举止（3课时）

- [ ] 第1课：站姿与坐姿 - 标准站姿训练、标准坐姿训练、走姿训练
- [ ] 第2课：表情与眼神 - 微笑训练、眼神交流、面部表情管理
- [ ] 第3课：手势与动作 - 恰当手势、避免小动作、整体协调

#### 语言表达（2课时）

- [ ] 第1课：语言表达技巧 - 语速与语调、停顿与重音、逻辑与条理
- [ ] 第2课：语言表达训练 - 口语化表达、避免口头禅、流畅度提升

#### 心理调适（2课时）

- [ ] 第1课：考前心理准备 - 自信心建立、紧张情绪缓解、积极心理暗示
- [ ] 第2课：考场心理调适 - 入场前调整、答题中调整、突发情况应对
